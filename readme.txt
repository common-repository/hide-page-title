== Description ==

Hide page title plugin is an useful and simple tool that allows to hide page and post titles. There are several options, that allow users to remove headlines from specific pages or just hide from all pages or posts.

In general settings there are two options - "Hide page titles from all pages" and "Hide post titles from all posts". Choose any option you need and just click the save button. If you need to remove headline for specific post, then go to the post from admin panel and edit it. You will find the headline option from the right bottom side of editor panel(bottom of Permalink, featured image and other options). 

**If you have any questions, issues or suggestions occurring this plugin contact us at [SUPPORT FORUM](https://wordpress.org/support/plugin/hide-page-title/).**

== Installation ==

**Download the plugin ZIP file.**  

1. Log in to WordPress admin panel.
2. Go to WordPress Plugins page, then click Add New, after that click Upload Plugin.
3. Click "Choose file" then select the plugin zip file.
4. After that Install and activate the plugin.

== Frequently Asked Questions ==

= Can I hide headlines from all pages/posts =

Yes, you can do it from admin panel, simply using two general options.
